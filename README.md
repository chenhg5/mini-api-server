# Mins

a mini restful server

to build a efficient restful server with only a command

## usage

```
mins -c /the/config/path
```

## route

| Method     | Path     |
| --------   | -----:   |
| GET        | /resource/:table/id/:id      |
| DELETE     | /resource/:table/id/:id      |
| PUT        | /resource/:table/id/:id      |
| POST       | /resource/:table             |
